## Task Management

To create a new task, use the command: **create task [description]**
To update an existing task, use: **update task [TASK_ID]**
To view tasks by filter, use: **show tasks [filter]** (all, active, pending, completed, blocked, recent)

Each task should have its own file in this directory with format: `TASKID-taskname.md`
